Attribute Methods of jQuerry

attr(properties) sets attribute value to all matched elements $("img").attr({alt:"Image not available", src:"img.png", title:"Image jQuery"});

attr(key, fn) sets a sing attribute or property to a value $("h1").attr("align", "center");

removeAttr(name) removes an attribute from all matched elements. $("h1").removeAttr("align", "center")

hasClass(Class find a specified CSS class is applied on any one of the matched elements, if yes, then it returns true otherwise false $("p#ptagid").hasClass("font")

addClass(class) used on elements to apply the specified class if it is not applied $("h1").addClass("font")

removeClass(class) used on elements to remove specified classes or class $("h1").removeClass("font")

html() to get first html tag or matched element $("h1").html();

text() to get text contents of all matched elements except form and input elements $("#id_div").text(title)

val() to get the value of input from the first matched element $("#inputid").val();
